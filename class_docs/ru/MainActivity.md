# Activity: MainActivity

## 1. Общая информация
*   **Имя класса:** `MainActivity`
*   **Тип:** Activity (Главный экран)
*   **Назначение:** Это «Центр управления». Обрабатывает основные действия пользователя: съемку фото, видео и переход к списку файлов или галерее.
*   **Взаимодействие с другими компонентами:** Запускает камеру, использует `DatabaseHelper` для записи, открывает экран просмотра списка.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение |
| :--- | :--- | :--- |
| `currentPhotoPath` | `String` | Хранит временный путь к сделанному фото. |
| `takePictureLauncher` | `ActivityResultLauncher` | Современный способ получения результата от камеры. |
| `dbHelper` | `DatabaseHelper` | Объект для связи с базой данных. |

## 3. Методы класса
### Имя метода: `dispatchTakePictureIntent`
*   **Что делает метод:** Создает пустой файл, подготавливает запрос к камере и запускает её, указывая, куда сохранить результат.
*   **Когда вызывается:** При нажатии на кнопку "Take Photo".

### Имя метода: `saveFileInfoToDatabase`
*   **Что делает метод:** Собирает имя файла, дату и время, а затем просит `dbHelper` сохранить их.

## 4. Жизненный цикл
*   **`onCreate()`**: Настраивает интерфейс, запрашивает разрешения (Камера и Память) и готовит «слушателей» для кнопок.

## 5. Взаимодействие с интерфейсом (UI)
*   **Кнопки:** Съемка фото, Съемка видео, Галерея, Инфо.
*   **События:** Обработка нажатий через `setOnClickListener`.

## 6. Взаимодействие с другими компонентами
*   **Системная камера:** Через `Intent`.
*   **База данных:** Через `DatabaseHelper`.

## 7. Общая логика класса
Этот класс координирует работу других приложений (камеры). Он сам не делает фото, а просит систему сделать это. Когда камера заканчивает работу, `MainActivity` получает сигнал и записывает данные в «дневник» (базу данных).

## 8. Упрощенное объяснение
**Объяснение простыми словами:**
Представьте `MainActivity` как **Прораба** на стройке. Прораб сам не кладет кирпичи и не водит грузовик (не делает фото). Вместо этого он вызывает «Специалиста по камере» и говорит: «Сделай свою работу и положи результат вот в эту коробку». Когда всё готово, Прораб записывает в журнал, что работа выполнена.
