# Activity: Start

## 1. Общая информация
*   **Имя класса:** `Start`
*   **Тип:** Activity (Экран)
*   **Назначение:** Это «Splash Screen» (заставочный экран) или точка входа в приложение. Он показывает временный экран (обычно с логотипом) перед тем, как перевести пользователя в основную часть приложения.
*   **Взаимодействие с другими компонентами:** Запускает `MainActivity` через некоторое время.

## 2. Переменные (поля класса)
В этом классе нет глобальных переменных. Вся логика находится внутри метода `onCreate`.

## 3. Методы класса
### Имя метода: `onCreate`
*   **Что делает метод:**
    1. Устанавливает макет экрана.
    2. Включает режим "Edge-to-Edge" (контент под статус-баром).
    3. Запускает таймер `CountDownTimer` на 2000 миллисекунд (2 секунды).
*   **Когда вызывается:** При первом открытии приложения.

### Имя метода: `onFinish` (внутри таймера)
*   **Что делает метод:**
    1. Создает `Intent` (запрос на переход) для `MainActivity`.
    2. Вызывает `startActivity(go)` для смены экрана.
*   **Когда вызывается:** Автоматически, когда таймер доходит до нуля.

## 4. Жизненный цикл
*   **`onCreate()`**: Вызывается при создании Activity. Здесь настраивается интерфейс и запускается таймер.

## 5. Взаимодействие с интерфейсом (UI)
*   **Макет:** Использует `R.layout.activity_start`.
*   **Адаптация:** Регулирует отступы, чтобы системные элементы (часы, индикатор батареи) не перекрывали контент.

## 6. Взаимодействие с другими компонентами
*   **MainActivity:** Переходит на этот экран через 2 секунды.

## 7. Общая логика класса
Приложение начинается здесь. Пользователь видит экран в течение 2 секунд. За кулисами идет обратный отсчет. Как только время истекает, приложение автоматически переключается на главный экран.

## 8. Упрощенное объяснение
**Объяснение простыми словами:**
Представьте класс `Start` как **зал ожидания** или **заставку перед фильмом**. Когда вы приходите в кинотеатр, фильм не начинается мгновенно — сначала показывают логотип киностудии. После этого начинается основное действие. Этот класс — та самая 2-секундная заставка.
