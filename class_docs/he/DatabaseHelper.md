# מחלקה: DatabaseHelper

## 1. מידע כללי
*   **שם המחלקה:** `DatabaseHelper`
*   **סוג:** מחלקה רגילה (יורשת מ-`SQLiteOpenHelper`)
*   **תפקיד באפליקציה:** מחלקה זו מנהלת את בסיס הנתונים המקומי (SQLite). היא אחראית על יצירת קובץ בסיס הנתונים, יצירת טבלאות, עדכון המבנה והכנסת נתונים.
*   **אינטראקציה עם רכיבים אחרים:**
    *   `MainActivity` משתמשת בה לשמירת מידע על קבצים חדשים.
    *   `ViewInfoAboutMyFiles` משתמשת בה לקריאת רשימת הקבצים.

## 2. משתנים (שדות המחלקה)
| שם | סוג | מטרה |
| :--- | :--- | :--- |
| `DATABASE_NAME` | `String` | שם הקובץ של בסיס הנתונים (`all_info.db`). |
| `DATABASE_VERSION` | `int` | מספר גרסה (עוזר בעדכונים). |
| `TABLE_NAME` | `String` | שם הטבלה השומרת את המידע. |
| `COLUMN_...` | `String` | שמות העמודות (שם, תאריך, זמן, נתיב, סוג). |
| `SQL_CREATE_ENTRIES` | `String` | פקודת ה-SQL ליצירת הטבלה. |

## 3. מתודות (שיטות) המחלקה
### שם המתודה: `onCreate`
*   **מה המתודה עושה:** מבצעת את פקודת ה-SQL ליצירת הטבלה.
*   **מתי נקראת:** באופן אוטומטי על ידי אנדרואיד בפעם הראשונה שהאפליקציה מנסה לגשת לבסיס הנתונים.

### שם המתודה: `insertFileInfo`
*   **מה המתודה עושה:**
    1. פותחת את בסיס הנתונים לכתיבה.
    2. אורזת את הערכים לאובייקט `ContentValues`.
    3. מכניסה את הנתונים לטבלה.
    4. סוגרת את החיבור.
*   **מתי נקראת:** באופן ידני מ-`MainActivity` לאחר צילום תמונה או וידאו.

## 4. מחזור חיים (Lifecycle)
*   מנוהל על ידי המערכת (מחזור החיים של בסיס הנתונים).

## 5. אינטראקציה עם הממשק (UI)
*   אין אינטראקציה ישירה עם ה-UI. היא עובדת "מתחת למכסה המנוע".

## 6. אינטראקציה עם רכיבים אחרים
*   **בסיס נתונים SQLite:** היא הגשר בין קוד הג'אווה לבין הקובץ האמיתי בזיכרון הטלפון.

## 7. לוגיקה כללית של המחלקה
מחלקה זו מרכזת את כל ה"רישום". במקום לכתוב קוד SQL מורכב בכל מקום באפליקציה, אנו פשוט קוראים למתודות פשוטות כמו `insertFileInfo`.

## 8. הסבר פשוט
**הסבר במילים פשוטות:**
חשבו על `DatabaseHelper` כעל **ספרן**. כשרוצים לרשום ספר (קובץ) חדש בספרייה, אתם לא הולכים וכותבים בספר הרישומים הגדול בעצמכם. אתם נותנים את המידע לספרן, והוא יודע בדיוק באיזה ספר (בסיס נתונים), באיזה דף (טבלה) ובאיזו עמודה לרשום אותו.
